IS INSTANCE


- O QUE É
		Is instance é uma função nativa do python que é usada para checar o tipo de um dado que ela recebe. Tem retorno 
		booleano, ou seja, caso um dado seja do tipo a ser checado, a função retorna 'True', caso contrário, retorna 
		'False'

- COMO USAR
		Para usar a função, é só passar dentro dela o dado a ser checado e o tipo de dado que é para checar:
						
							a = 5	# Inteiro, int
							b = isinstance(a, int)	# Retorno 'True'
							c = isinstance(a, str)	# Retorno 'False'
